# 전체 검증 (서브에이전트 실행)

순서대로 실행하고 에러를 자동 수정합니다:

1. `pnpm typecheck` 실행
2. 에러 있으면 → 원인 분석 → 수정 → 재실행
3. `pnpm lint` 실행
4. 에러 있으면 → `pnpm lint --fix` → 수동 수정 필요한 항목 처리
5. `pnpm build` 실행
6. 에러 있으면 → 원인 분석 → 수정 → 재실행

각 단계 최대 5회 반복. 5회 초과 시 에러 목록 + 시도한 해결책 보고.

## 최종 보고 형식
```
✅ typecheck: PASS (에러 0)
✅ lint: PASS (에러 0, 경고 N)
✅ build: PASS
수정된 파일: [목록]
```

에러가 남아있으면 "완료" 대신 남은 에러 목록을 보고합니다.
