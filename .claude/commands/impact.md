# 영향도 전수 검색

$ARGUMENTS 에 해당하는 키워드(함수명, 타입명, import 경로, 컴포넌트명)를 프로젝트 전체에서 검색합니다.

## 수행 작업

1. 주어진 키워드를 `grep`으로 전체 프로젝트 검색 (src/, docs/ 포함)
2. 발견된 모든 파일을 분류:
   - **직접 사용**: import/호출하는 파일
   - **간접 참조**: 타입 참조, re-export
   - **문서**: 주석, README 등
3. 영향 범위 시각화:
   ```
   변경 대상: [키워드]
   ├── 직접 영향: [파일 목록]
   ├── 간접 영향: [파일 목록]
   └── 문서 업데이트 필요: [파일 목록]
   ```
4. 각 파일별 업데이트 필요 여부 판단

인자가 없으면 가장 최근 변경된 파일에서 export된 심볼을 자동으로 검색합니다.
